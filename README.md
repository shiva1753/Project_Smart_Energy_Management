# Smart Home Energy Management System

An ongoing capstone project focused on monitoring household energy consumption using IoT devices and Machine Learning.

## Overview

The Smart Home Energy Management System aims to provide real-time monitoring and analysis of appliance-level power consumption. The system integrates IoT hardware, MQTT communication, a FastAPI backend, and a React dashboard to help users track and understand their energy usage.

## Current Features

* Real-time energy data collection from connected devices.
* MQTT-based communication between hardware and backend services.
* FastAPI backend for data processing and API management.
* Interactive dashboard for monitoring energy consumption.
* Energy consumption forecasting using a Random Forest Regression model.

## Tech Stack

* React.js
* FastAPI
* MQTT
* Scikit-learn
* Raspberry Pi

## Project Status

🚧 Currently under development as part of an ongoing capstone project.

Planned enhancements include improved analytics, anomaly detection, and advanced forecasting capabilities.
